# AI4NTH Summer School GitHub Pages Site

This repository contains the GitHub Pages website for the **AI for Nuclear Thermal Hydraulics Summer School**.

## Pages

- `index.md` — landing page
- `programme.md` — programme and timetable
- `materials.md` — overview of teaching materials and repository structure
- `lecturer_guidance.md` — lecturer guidance
- `_layouts/default.html` — site layout with navigation bar
- `assets/style.css` — custom styling
- `assets/ccp-nth-logo-new-long.png` — editable placeholder logo

## Repository Structure

This repository contains both the event website and the reusable training materials.

- `programme.md` — public programme page
- `lecturer_guidance.md` — guidance for invited lecturers
- `environment/` — Python/Jupyter environment setup
- `datasets/` — dataset documentation and split files
- `course_materials/` — slides, notebooks, exercises, and solutions organised by day
- `instructors/` — templates and checklists for contributors
