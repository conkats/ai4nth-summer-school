# Course Materials

This folder contains teaching materials for the AI4NTH Summer School, organised by course day.

Released course files are kept directly in the relevant day folder so students can find notebooks, slides, PDFs, and HTML tools without drilling into nested folders. The `exercises/` and `solutions/` folders remain available for structured practical tasks and reference outputs.

Materials should be reproducible, lightweight enough for standard laptops, and documented with expected practical outputs and any additional software requirements.
