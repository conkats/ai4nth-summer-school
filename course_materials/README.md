# Course Materials

This folder contains the day-by-day learning materials for the AI4NTH Summer School.

## How Students Should Use This Folder

Start with Day 0 after your environment check has passed:

1. Open `day0_onboarding/`.
2. Read `day0_onboarding/README.md`.
3. Run `day0_onboarding/Day0_self_learning_onboarding.ipynb` in JupyterLab.
4. During the school, follow the day folder named by the course team.

Released notebooks, slides, PDFs, and HTML tools are kept directly in each day folder so they are easy to find. Some days may also include `exercises/` or `solutions/` folders when separate practical tasks or reference outputs are released.

## Day Folders

- `day0_onboarding/` - setup check, dataset orientation, and terminology.
- `day1_foundations/` - baseline workflow and foundational modelling material.
- `day2_tree_models/` - tree-based models.
- `day3_surrogates_gp/` - surrogate modelling and Gaussian processes.
- `day4_neural_networks/` - neural networks.
- `day5_vvuq_optimisation/` - VVUQ and optimisation.

## For Lecturers

This section is for lecturers and organisers preparing materials.

Keep notebooks runnable on standard laptops, avoid unnecessary dependencies, and document expected practical outputs and any additional software requirements in the relevant day README.
