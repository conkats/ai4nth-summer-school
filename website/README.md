# Website Pages

Students should normally use the public website links from the root README rather than editing files in this folder.

This folder contains the source files used to build the public GitHub Pages site for the AI4NTH Summer School.

## For Website Maintainers

This section is for organisers and maintainers updating the public website.

The Markdown files use explicit Jekyll permalinks so the public website remains available at the root-level URLs, for example `/programme.html` and `/registration.html`, even though the source files are grouped here.

Website support files remain in their standard Jekyll locations:

- `_layouts/` - shared page layout
- `assets/` - images and CSS
